1. Download and Unzip the folder
2. Extract the zip into your xampp htdocs folder 
3. Add the databse in the phpmyadmin
4. Update the site url based on the folder extracted, Update the link in wp_options table
5. After Setitng up ,open the site using http://localhost/megamart
6. For accessing the admin panel using http://localhost/megamart/wp-admin/
